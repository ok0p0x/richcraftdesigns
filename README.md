# Rich Craft Designs Vercel Site

This is a clean Next.js site for Vercel. It uses no external media dependencies, no shadcn, no Tailwind, and no extra libraries.

## Local development

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## Replace placeholder images

Add your real project photos to:

```txt
public/projects/
```

Use these file names first, or update app/page.jsx:

- deck-box.jpg
- slate-coasters.jpg
- layered-art.jpg
- metal-sign.jpg
- cabin-gift.jpg
- workshop-video.jpg

If those files do not exist, the site still works and shows emoji placeholders.

## Deploy on Vercel

Upload this project to GitHub, then import the repo in Vercel.
